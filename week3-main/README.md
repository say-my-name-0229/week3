## 📌 Overview

This repository contains the implementation of **React.js Lab – Experiment 3**, demonstrating the fundamental concepts of React.js using functional components and Hooks.

The experiment includes practical applications of:

- State Management using **useState**
- Side Effects using **useEffect**
- Component Communication using **Props**
- Controlled Forms
- Dynamic Rendering using **map()**

These programs are designed to provide hands-on experience with modern React development.
